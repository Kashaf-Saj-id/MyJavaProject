Package: AssessmentPackage

1. Assessment.java:
   - Main class demonstrating various Java features such as loops, string manipulation, method overloading, and method overriding.

2. Calcu.java:
   - Class demonstrating method overloading by providing multiple `add()` methods for adding two, three, or four integers.

3. Wild.java:
   - Base class with a `sound()` method representing the sound made by wild animals.

4. Sparrow.java:
   - Subclass of Wild that overrides the `sound()` method to represent the sound made by a Sparrow.
